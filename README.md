# JavaScript Projects Suite

A collection of mini web applications built with HTML, CSS, and JavaScript.

## 🚀 Projects Included

| Project | Description | Main Tech |
| :--- | :--- | :--- |
| **[Timer](Timer/index.html)** | A customizable countdown timer with quick-add presets. | JS, CSS Flexbox |
| **[Stopwatch](stopwatch/index.html)** | A precise stopwatch with start, stop, and reset functionality. | JavaScript DOM |
| **[World Checker](worldchecker/index.html)** | A text analysis tool (Word/Character counter) with interactive UI. | JS, CSS Animations |
| **[Counter](counter/index.html)** | A simple and clean number counter. | Basic JS |
| **[Random Quotes](RandomQutes/index.html)** | Displays inspiration via randomly generated quotes. | JavaScript |

## 🛠️ Features

- **Responsive Design:** All tools are designed to work across different screen sizes.
- **Modern UI:** Clean, "classy" interfaces with smooth transitions and hover effects.
- **Standalone:** Each project is contained in its own directory for easy exploration.

## 📂 Project Structure

```text
projests/
├── Timer/          # Countdown timer app
├── stopwatch/      # Stopwatch app
├── worldchecker/   # Text analysis tool
├── counter/        # Basic counter
└── RandomQutes/    # Quote generator
```

## 📝 How to Run

Simply open the `index.html` file within any project folder in your favorite web browser.
